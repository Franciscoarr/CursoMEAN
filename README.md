# CRUD de Productos - Stack MEAN

Este es un proyecto desarrollado durante el curso **"CRUD Angular - Node - MEAN"** en Udemy
Se trata de una aplicación full-stack que permite gestionar un inventario de productos, poniendo en práctica la integración de un frontend en Angular con un backend en Node.js y MongoDB

## Tecnologías Utilizadas

### Frontend
* **Angular**: Framework principal
* **Bootstrap**: Diseño responsivo y estilos
* **Toastr**: Notificaciones interactivas
* **Google Fonts**: Tipografías personalizadas
* **Formularios Reactivos**: Para la validación y captura de datos

### Backend
* **Node.js**: Entorno de ejecución
* **Express**: Framework para la creación de la API REST
* **MongoDB**: Base de datos NoSQL en la nube
* **Mongoose**: Modelado de objetos para MongoDB
* **CORS / Dotenv**: Configuración de seguridad y variables de entorno
